Banco de Dados

A disciplina trabalha com os conceitos da teoria de banco de dados envolvendoa spectos da arquitetura, modelagem conceitual com ênfase ao modelo relacional, através de uma linguagem para definição e manipulação de dados.

Objetivos

  • Permitir ao aluno assimilar conhecimentos fundamentais em banco de dados,incluindo modelos de dados,arquitetura de SGBDs, acesso a banco de dados, segurança, integridade.
  • Capacitar o aluno a projetar banco de dados relacionais para aplicações e compreender os princípios de organização dos dados.
  • Habilitar o aluno a criar fisicamente esses banco de dados sobre SGBDs relacionais e acessá-los de maneira adequada via linguagemSQL.

Conteúdo Programático

  • Introdução a banco de dados:abstração de dados;
  • Modelagem de dados:etapas do projeto de um banco de dados relacional, modelo Entidade - Relacionamento(ER), mapeamento ER-relacional, construções básicas do DER, extensões do DER;
  • Teoria da normalização: objetivo, dependências funcionais, formas normais(1FN, 2FN, 3FN e FNBoyce-Codd);
  • Introdução ao SQL: comando de DDL(DataDefinitionLanguage), Comandos de DML(DataManipulationLanguage): conceitos básicos de consulta SQL,restrições de integridade, visões, autorização de acesso;
  • Linguagem de Manipulação de Dados: Insert, Select, funções de agregação, atualização de dados, conceito de Joins, conceito de Stored Procedure e conceito de Trigger.
